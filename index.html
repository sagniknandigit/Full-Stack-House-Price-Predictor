About This Project

This project showcases a **full-stack machine learning application** designed to estimate house prices. Built using Python's Scikit-learn for the predictive model and a Flask API for backend integration, it provides a seamless user experience.

The frontend, crafted with modern HTML and styled beautifully with Tailwind CSS, allows users to input various house features and instantly receive a price prediction. It's a testament to deploying ML models from concept to interactive web solution!

Tools: Python, Scikit-learn, Pandas, Flask, HTML, CSS (Tailwind), JavaScript
